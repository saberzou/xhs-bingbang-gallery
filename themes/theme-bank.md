# Theme Bank for @BingBang Daily Content

## How This Works
Each theme has:
- `text`: the English phrase shown at the bottom of the image (short, punchy, ends with period)
- `objects`: the cute doodle objects to draw (drives the image prompt)
- `caption`: the Chinese XHS post caption

All images follow the same style: cute kawaii objects with faces, thick black outlines,
solid black fills for contrast, 4-pointed sparkle stars (✦) scattered around, white background.

## Rotation Rule
Week pattern: 设计→生活→创意→文字→情绪→设计→生活
Pick themes randomly within each pillar for variety.
Never repeat the same theme within a 2-week window.
